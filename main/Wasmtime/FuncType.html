Class Method Details

.new(params, results) ⇒ FuncType

A descriptor for a function in a WebAssembly module. WebAssembly functions can have 0 or more parameters and results. Each param must be a valid WebAssembly type represented as a symbol. The valid symbols are: :i32, :i64, :f32, :f64, :v128, :funcref, :externref.

Examples:

FuncType that takes 2 i32s and returns 1 i32:

FuncType.new([:i32, :i32], [:i32])
